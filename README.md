### ğŸ§­ Project Workflow

The project is structured into four main phases, mirroring a real-world data science workflow:

1. **Web Scraping**  
   Collected over 900 smartphone listings from Flipkart using `Python`, `Requests`, and `BeautifulSoup` to build a dataset reflecting current market conditions.

2. **Data Cleaning & Feature Engineering**  
   Processed the raw scraped data using `pandas`.  
   Extracted and standardized features like RAM, Storage, Display, Processor, and Battery using regex and string processing.

3. **Machine Learning Modeling**  
   Built and optimized a prediction model using `scikit-learn`â€™s `RandomForestRegressor` within a pipeline.  
   Hyperparameter tuning was performed using `RandomizedSearchCV` to improve accuracy.

4. **Deployment**  
   Created an interactive web interface with `Streamlit`.  
   Users can adjust smartphone specifications and receive a **real-time price prediction** generated by the saved ML model.
